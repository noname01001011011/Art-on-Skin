INK STUDIO — Modern Tattoo Studio Website

A fully responsive, multi‑language, production‑grade website built with clean HTML, SCSS, and vanilla JavaScript.
Designed and developed with a strong focus on UI/UX, accessibility, performance, and maintainability.

🚀 Overview
INK STUDIO is a modern landing website for a tattoo studio, featuring:

a polished, premium visual style

adaptive layout with strict breakpoints

interactive components (gallery filters, sliders, modal booking form)

multilingual support (i18n)

clean BEM‑structured SCSS

optimized images (WebP, lazy loading)

smooth animations and transitions

form validation with user‑friendly feedback

This project demonstrates my ability to design, structure, and implement a complete commercial‑grade website from scratch — including UI/UX decisions, layout architecture, component logic, and performance optimization.

🎨 Features
✔ Fully Responsive Layout
Built with strict breakpoints (1025px, 769px, 426px, 376px, 321px) ensuring pixel‑perfect rendering across:

desktop

tablets

mobile devices

Every section adapts harmoniously without breaking visual hierarchy.

✔ Interactive Gallery with Filters
A dynamic gallery with:

category filtering

fade + slide animations

staggered appearance

dynamic grid restructuring (1 / 2 / 3 columns)

automatic centering of filtered images

No external libraries — pure JavaScript.

✔ Reviews Slider (Swiper.js)
A custom‑styled slider with:

external positioned arrows

clean integration with the site’s visual language
